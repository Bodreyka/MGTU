Вариант № 3 "Модель процесса электронно-лучевой сварки"

Задача: Предсказание ширины и глубины сварного шва основании параметров 
		настройки электронно-лучевой сварки.
Решение:Консольная программа с GUI интерфейсом. Прогноз строится с 
		использованием предобученной нейронной сети. (Папка: src)

Доп. материалы: В формате  Jupyter Note Book приложенны файлы:
..notebook/regression_classic.ipynb - первичный анализ датасета, попытка решения класическими методами регресси:
									метод наименьших квадратов, случайный лес, метод ближайших соседей,
									метод опорных векторов.
..notebook/regression_neyrone.ipynb  - решение с использованием нейронной сети. Проведен ручной подбор параметров и сохранение.
..notebook/normaliser.ipynb			 - предобработка данных и сохранение обученных объектов для использования в консольном приложении.
